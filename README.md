# FD Adventure Toolkit
Extensible Tools for Creating Point'n'Click Adventure Games and state games in general.

## How to setup
- In Unity - open Package Manager
- Click '+' Button and choose 'Install package from git URL'
- Paste this URL in: `https://github.com/filipdufka/fd_adventure_toolkit.git`
## How to use
- Add Component Button from FD.AdventureToolkit library
- Make sure you have some of the Collider2D on the game object
- In the Button, create binding for Click Action with '+' button and set it mostly to 'Left Button [Mouse]'
- Create reaction to this click with On Click Unity Event
